
Welcome to segment-lidar's documentation!
=========================================

About
-----

The package **segment-lidar** is specifically designed for unsupervised instance segmentation of aerial LiDAR data. It brings together the power of Segment-Anything Model (SAM) developed by Meta Research and segment-geospatial (SamGeo) package from Open Geospatial Solutions to automatize the segmentation of LiDAR data.
If you use this package for your research, please cite:
